### Setting Up The ChatBot
- **Download the chatbot folder and user_ui folder**
- **Create a new folder**
- **Place the two files in the folder**
- **In the folder, create a venv folder and activate your virtual environment**
- **Open the folder in VS code or your code editor**
- **Making sure your virtual environment is activate, run this command `pip install -r requirements.txt` to download all the packages and dependencies**
- **Navigate your current directory using the `cd` command to the _chatbot_ folder**
- **Run the backend using `uvicorn --reload gemini_chat:app`**
- **Make sure you have the LiveServer extension**
- **Open the *user_ui* folder which contains the frontend, click on the `chat.html` file, right click from inside the file and _Open with LiveServer_**
